Site Web Portfolio

Ce projet a consisté en la création de mon site web personnel responsive pour présenter mes compétences, mon parcours et mes réalisations professionnelles. L'objectif était de fournir une plateforme centralisée et visuellement attrayante pour mon portfolio en ligne.

Technologies Utilisées

  • HTML5
  • CSS3
  • JavaScript (pour la navigation par onglets)
  • Font Awesome (pour les icônes)

Fonctionnalités Clés

  • Design entièrement responsive : compatible avec tous les types d’appareils (mobiles, tablettes, ordinateurs)
  • Navigation fluide par onglets dynamiques : sections accessibles sans rechargement de page (Accueil, À Propos, Projets, Bilan, Contact).
  • Formulaire de contact
  • Sections dédiées : Compétences techniques et transversales, Parcours académique et professionnel, Centres d’intérêt et passions
  • Animations CSS pour une meilleure expérience utilisateur.

Défis et Solutions

L’un des principaux défis a été de garantir une compatibilité optimale sur tous les appareils, en adaptant soigneusement la structure CSS pour assurer un affichage fluide quel que soit le support utilisé. J’ai également développé une logique JavaScript personnalisée pour permettre le changement d’onglets sans recharger la page, améliorant ainsi la fluidité et la réactivité de la navigation.

Enfin, l’utilisation d’icônes via Font Awesome a permis d’enrichir visuellement l’interface et de rendre le contenu plus clair et professionnel.
